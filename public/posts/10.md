## JavaScript란 무엇일까

JavaScript는 **웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어**입니다.  
쉽게 말하면, **사용자가 버튼을 클릭했을 때 반응하거나, 데이터를 계산하는 등 "움직이는" 기능**을 담당합니다.

HTML과 CSS만으로는 웹 페이지가 정적입니다. 하지만 JavaScript를 추가하면:
- 버튼 클릭 시 특정 동작 실행
- 사용자 입력값 검증
- 실시간 데이터 업데이트
- 게임이나 인터랙티브한 기능 구현

이런 모든 **동적인 동작**을 JavaScript가 담당합니다.

---

## HTML, CSS, JavaScript의 역할 분담

웹 개발의 3가지 핵심 기술을 이해하면 개발이 훨씬 쉬워집니다.

**HTML (구조)**: 웹 페이지가 무엇을 보여줄 것인가?
- 제목, 문단, 이미지, 입력 폼 등 정의

**CSS (스타일)**: 웹 페이지가 어떻게 보일 것인가?
- 색상, 크기, 위치, 애니메이션 등 정의

**JavaScript (동작)**: 웹 페이지가 어떻게 반응할 것인가?
- 클릭, 입력, 시간 경과 등의 이벤트에 반응
- 데이터 처리 및 계산
- 화면 업데이트

세 가지가 모두 협력하여 완성도 높은 웹사이트가 됩니다.

---

## JavaScript가 필요한 이유

현대 웹사이트는 매우 복잡하고 인터랙티브합니다.

- **검색 엔진**: 검색어 입력 → 실시간 검색 제안
- **소셜 미디어**: 좋아요 클릭 → 즉시 숫자 증가
- **온라인 쇼핑**: 상품 필터 → 실시간 목록 업데이트
- **게임**: 마우스/키보드 입력 → 실시간 게임 진행

이 모든 기능이 JavaScript로 구현됩니다.

---

## JavaScript의 실행 위치

JavaScript는 보통 `<script>` 태그를 사용하여 HTML에 포함됩니다.

### 방법 1: 외부 파일 연결 (권장)

HTML:

```html
<script src="script.js"></script>
```

### 방법 2: HTML 내부에 작성

```html
<script>
  console.log("Hello, JavaScript!");
</script>
```

### 방법 3: HTML 요소에 직접 작성

```html
<button onclick="alert('버튼 클릭됨')">클릭하세요</button>
```

일반적으로는 외부 파일로 관리하는 것이 가장 좋습니다.

---

## 간단한 JavaScript 예제

버튼 클릭 시 메시지를 표시하는 예제입니다.

HTML:

```html
<button id="myButton">클릭하세요</button>
```

JavaScript:

```javascript
const button = document.getElementById("myButton");

button.addEventListener("click", function() {
  alert("버튼이 클릭되었습니다!");
});
```

이 코드는:
1. HTML에서 ID가 "myButton"인 버튼을 찾음
2. 그 버튼에 클릭 리스너를 추가
3. 버튼이 클릭되면 알림창을 띄움

## JavaScript를 배워야 하는 이유

JavaScript를 배우면:
- 웹사이트의 모든 동적 기능을 구현할 수 있음
- 프론트엔드 개발의 기초가 됨
- 취업 시장에서 수요가 매우 많음
- 다른 언어를 배우기가 훨씬 쉬워짐

HTML과 CSS를 배웠다면, JavaScript는 필수 과정입니다!
