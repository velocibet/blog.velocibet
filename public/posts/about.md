## 소개

웹 개발을 취미로 하고 있습니다. HTML, CSS, JavaScript를 시작으로 웹 기술들을 배우고 실습하고 있습니다. 주로 Nuxt + NestJS를 사용하는 것을 좋아합니다.

## 현재 진행 중인 프로젝트

현재 **웹 기반 실시간 메신저**를 개발 중입니다. WebSocket을 활용한 실시간 채팅, 사용자 인증, 파일 전송 등 다양한 기능을 구현하고 있으며, 향후 데스크톱 애플리케이션과 모바일 앱으로도 확장할 계획입니다.

**메신저 사이트**: [https://www.velocibet.com/](https://www.velocibet.com/)

## 이 블로그 소개

이 블로그는 제가 웹 개발을 공부하면서 배운 내용들을 정리하고, 프로젝트 진행 과정을 기록하는 공간입니다.

### 블로그 카테고리

- **개발 일지**: 메신저 프로젝트의 개발 과정과 기술 스택 정리
- **웹개발**: HTML, CSS, JavaScript 기초부터 심화까지의 강좌
