## 선택자(Selector)란?

선택자는 **CSS 스타일을 적용할 HTML 요소를 선택하는 방법**입니다.  
같은 스타일을 여러 요소에 적용하거나, 특정 요소만 선택하여 다른 스타일을 주는 등 매우 유용합니다.

---

## 기본 선택자들

### 1. 태그 선택자 (Type Selector)

모든 같은 태그 요소에 스타일을 적용합니다.

```css
h1 {
  color: blue;
}

p {
  font-size: 16px;
}
```

이렇게 하면 **모든 `<h1>` 태그가 파란색**, **모든 `<p>` 태그의 글자가 16px**가 됩니다.

### 2. 클래스 선택자 (Class Selector)

`.` 기호를 사용하여 특정 클래스를 가진 요소에 스타일을 적용합니다.

HTML:

```html
<p class="highlight">이 문단은 강조됩니다.</p>
<p>이 문단은 일반입니다.</p>
```

CSS:

```css
.highlight {
  background-color: yellow;
  font-weight: bold;
}
```

결과: 첫 번째 문단만 노란 배경에 굵은 글씨입니다.

### 3. ID 선택자 (ID Selector)

`#` 기호를 사용하여 특정 ID를 가진 요소에 스타일을 적용합니다.

HTML:

```html
<div id="header">헤더 영역</div>
```

CSS:

```css
#header {
  background-color: gray;
  padding: 20px;
}
```

---

## 선택자의 특이성(Specificity)

같은 요소에 여러 선택자가 적용될 때, **더 구체적인 선택자가 우선**됩니다.

우선순위:
1. **ID 선택자** (가장 높음)
2. **클래스 선택자**
3. **태그 선택자** (가장 낮음)

```css
p {
  color: blue;           /* 우선순위: 낮음 */
}

.important {
  color: red;            /* 우선순위: 중간 */
}

#main {
  color: green;          /* 우선순위: 높음 → 최종 적용 */
}
```

같은 ID를 가진 `<p>`에는 초록색이 적용됩니다.

---

## 여러 선택자 함께 사용하기

### 그룹 선택자

여러 요소에 같은 스타일을 적용할 때는 쉼표로 구분합니다.

```css
h1, h2, h3 {
  color: navy;
  margin-bottom: 15px;
}
```

### 자식 선택자

특정 요소 안의 자식 요소만 선택합니다.

```css
div > p {
  color: blue;  /* div 안의 p만 파란색 */
}
```

## 선택자를 잘 이해하는 것의 중요성

선택자를 효과적으로 사용하면:
- 코드 중복을 줄일 수 있습니다
- 유지보수가 쉬워집니다
- 더 정교한 디자인을 할 수 있습니다

HTML의 구조를 이해하고 선택자를 적절히 선택하는 것이 CSS를 잘하는 첫 걸음입니다.
