## HTML이란 무엇일까

HTML은 **HyperText Markup Language**의 줄임말로, 웹 페이지를 구성하는 기본 언어입니다.  
쉽게 말하면, **웹 페이지의 뼈대**를 만드는 도구라고 할 수 있어요.

HTML을 이용하면 글, 이미지, 링크 같은 콘텐츠를 브라우저가 이해할 수 있는 형태로 작성할 수 있습니다.  
예를 들어, 문단을 나누거나 제목을 표시하고, 이미지를 삽입하는 것도 HTML이 담당합니다.

---

## HTML 문서의 기본 구조

모든 HTML 문서는 비슷한 구조를 가지고 있습니다.  
가장 기본적인 틀은 이렇게 생겼어요.

```html
<!DOCTYPE html>
<html>
  <head>
    <title>웹 페이지 제목</title>
  </head>
  <body>
    웹 페이지에 보여줄 내용
  </body>
</html>
```

- `<!DOCTYPE html>`: HTML 문서임을 브라우저에 알려주는 선언입니다.
- `<html>`: HTML 문서의 시작과 끝을 나타냅니다.
- `<head>`: 문서의 정보(metadata)를 넣는 곳입니다. 제목, 문자 인코딩, 스타일 시트 연결 등이 여기에 포함됩니다.
- `<body>`: 실제로 화면에 보여질 콘텐츠가 들어가는 영역입니다.

## HTML의 핵심 개념

HTML은 **태그(Tag)**로 이루어져 있습니다.
태그는 `<태그이름>내용</태그이름>` 형태로 쓰며, 내용이 어떤 용도인지 브라우저에게 알려주는 역할을 합니다.

예를 들어:

- `<h1> ~ <h6>`: 제목을 나타냅니다. 숫자가 작을수록 큰 제목
- `<p>`: 문단(Paragraph)을 나타냅니다.

HTML은 어렵지 않습니다. 중요한 것은 **내용을 구조화하는 언어**라는 점이에요.
