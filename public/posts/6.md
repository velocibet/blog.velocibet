## HTML 주석이란?

웹 페이지를 만들다 보면 코드 안에 **설명이나 메모**를 남겨야 할 때가 있습니다.  
이때 사용하는 것이 바로 **주석(Comment)**입니다.  
HTML 주석은 브라우저에 표시되지 않고, 오직 **개발자만 확인할 수 있는 메모** 역할을 합니다.

주석은 코드의 가독성을 높이고, 협업할 때 다른 개발자에게 의도를 전달하거나 나중에 자신이 이해하기 쉽게 기록할 수 있는 중요한 도구입니다.  
또한, 일시적으로 코드 일부를 실행되지 않게 하고 싶을 때도 주석을 활용할 수 있어요.  

---

## 주석 작성 방법

HTML에서 주석은 **`<!--`로 시작하고 `-->`로 끝납니다.**  
예를 들어, 아래와 같이 작성할 수 있습니다.

```html
<!-- 이 부분은 헤더 영역입니다 -->
<header>
  <h1>나의 첫 웹 페이지</h1>
</header>
```

- `<!--`와 `-->` 사이의 내용은 브라우저에 표시되지 않습니다.
- HTML 주석 안에는 어떤 내용이든 자유롭게 작성할 수 있습니다.
- 단, `--` 문자는 연속으로 사용하지 않는 것이 좋습니다. HTML 표준에서 허용되지 않기 때문이에요.

---

## 주석을 활용하는 이유

### 설명 추가

코드의 목적이나 동작을 적어두면, 나중에 다시 보거나 다른 사람이 볼 때 이해하기 쉽습니다.

```html
<!-- 여기서 nav 메뉴를 시작합니다 -->
<nav>
  <ul>
    <li><a href="/">홈</a></li>
    <li><a href="/about">소개</a></li>
  </ul>
</nav>
```

### 코드 임시 비활성화

테스트할 때 특정 코드만 잠시 실행되지 않게 하고 싶을 때 주석을 활용합니다.

```html
<!-- <p>이 문장은 잠시 보이지 않습니다.</p> -->
```

### 협업에서 가이드 제공

여러 명이 함께 개발할 때, 특정 구조나 규칙을 주석으로 남기면 혼동을 줄일 수 있습니다.

---

## 주의할 점

- 주석을 너무 많이 남기면 코드가 길어지고 가독성이 오히려 떨어질 수 있습니다.
- 브라우저에는 표시되지 않지만, 페이지 소스를 보면 누구나 볼 수 있으므로 비밀번호나 민감한 정보는 주석에 절대 넣지 않아야 합니다.
- 주석은 어디까지나 설명용, 임시 비활성화용으로 사용해야 합니다.

HTML 주석은 간단하지만 효율적인 개발 습관을 만드는 데 중요한 도구입니다.
앞으로 HTML을 더 많이 작성할수록, 주석을 적절히 활용해 코드 관리와 협업에 큰 도움을 받을 수 있어요.
