## style 속성이란?

웹 페이지에서 HTML 요소에 **직접 CSS를 적용**하고 싶을 때 사용하는 속성이 `style`입니다.  
`style` 속성 안에 CSS 속성과 값을 적어주면, 해당 요소에만 스타일이 적용돼요.  

예를 들어 글자 색을 바꾸거나, 글자 크기를 바꾸는 것도 가능합니다.

---

## color와 font-size

- **color**: 글자의 색상을 바꿉니다.  
  - 예: `color: red;`, `color: #00ff00;`, `color: blue;`  

- **font-size**: 글자의 크기를 바꿉니다.  
  - 예: `font-size: 20px;`, `font-size: 1.5em;`  

이 두 가지 속성만 알아도 텍스트를 예쁘게 꾸미는 기본적인 작업은 충분히 할 수 있어요.

---

## p 태그에 적용하기

HTML에서 문단을 나타내는 `<p>` 태그에 style 속성을 적용해볼게요.

```html
<p style="color: blue; font-size: 18px;">
  안녕하세요! 이 문장은 파란색이고 글자 크기는 18px입니다.
</p>

<p style="color: red; font-size: 24px;">
  이번 문장은 빨간색이고 글자 크기는 24px로 표시됩니다.
</p>
```

- 첫 번째 문단: 글자 색상 파랑, 크기 18px
- 두 번째 문단: 글자 색상 빨강, 크기 24px

style 속성은 HTML 요소 하나하나에 바로 스타일을 적용할 수 있어서 간단한 테스트나 예제에 유용합니다.
다만 여러 요소에 같은 스타일을 적용하거나 유지보수가 필요할 때는 외부 CSS 파일이나 `<style>` 태그를 사용하는 것이 더 효율적이에요.
