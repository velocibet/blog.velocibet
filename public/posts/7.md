## CSS란 무엇일까

CSS는 **Cascading Style Sheets**의 줄임말로, 웹 페이지의 **디자인과 레이아웃**을 담당하는 언어입니다.  
쉽게 말하면, **HTML이 만든 뼈대에 색깔, 크기, 배치 같은 스타일을 입히는 도구**라고 할 수 있어요.

HTML만으로는 흰색 배경에 검은 글자뿐이지만, CSS를 사용하면:
- 글자 색깔과 크기 변경
- 배경색이나 배경 이미지 추가
- 요소들의 위치와 크기 조정
- 마우스 호버 효과 추가

이런 모든 시각적인 꾸밈을 CSS가 담당합니다.

---

## HTML과 CSS의 관계

HTML과 CSS는 서로 다른 역할을 합니다.

**HTML**: 웹 페이지의 **구조와 내용**
- 제목, 문단, 이미지, 링크 등의 요소를 정의
- "무엇을" 보여줄 것인가에 초점

**CSS**: 웹 페이지의 **스타일과 레이아웃**
- HTML 요소들을 어떻게 꾸밀 것인가 정의
- "어떻게" 보여줄 것인가에 초점

예를 들어, `<h1>제목</h1>`은 HTML이 만들고, 이 제목의 색깔을 파란색으로, 글자 크기를 30px로 만드는 것은 CSS입니다.

---

## CSS가 중요한 이유

현대 웹사이트는 모두 아름답고 세련된 디자인을 가지고 있습니다. 이것이 가능한 이유는 CSS 때문입니다.

- **사용자 경험 향상**: 보기 좋은 디자인은 방문자를 오래 머물게 함
- **브랜드 아이덴티티**: 일관된 색상과 폰트로 브랜드 이미지 구축
- **반응형 디자인**: 모바일, 태블릿, 데스크탑 등 다양한 화면 크기에 대응

CSS를 잘 다루면, HTML과 조합하여 정말 멋진 웹사이트를 만들 수 있습니다.

---

## CSS의 기본 형태

CSS는 매우 간단한 구조를 가지고 있습니다.

```css
선택자 {
  속성: 값;
  속성: 값;
}
```

예를 들어:

```css
h1 {
  color: blue;
  font-size: 24px;
}
```

- **선택자(selector)**: 스타일을 적용할 HTML 요소 (예: h1, p, div)
- **속성(property)**: 무엇을 꾸밀 것인가 (예: color, font-size)
- **값(value)**: 어떻게 꾸밀 것인가 (예: blue, 24px)
