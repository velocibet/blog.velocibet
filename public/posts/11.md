## 변수란?

변수는 **데이터를 저장하는 상자**라고 생각하면 됩니다.  
프로그래밍에서는 숫자, 텍스트, 참/거짓 등의 데이터를 변수에 저장하여 나중에 사용합니다.

```javascript
let name = "Velocibet";  // 변수 name에 "Velocibet" 저장
console.log(name);        // "Velocibet" 출력
```

---

## 변수 선언하기

JavaScript에서 변수를 선언하는 방법은 세 가지입니다.

### 1. let (권장)

```javascript
let age = 25;
age = 26;  // 값 변경 가능
```

`let`은 현대적인 방식으로, 블록 스코프를 가집니다. 대부분의 경우 이것을 사용합니다.

### 2. const (권장)

```javascript
const pi = 3.14159;
// pi = 3.14;  // 오류! 값 변경 불가능
```

`const`는 한 번 저장하면 변경할 수 없습니다. 변경되지 않을 데이터는 `const`로 선언하세요.

### 3. var (구형, 지양)

```javascript
var score = 100;
score = 95;  // 값 변경 가능
```

`var`는 예전 방식입니다. 새로운 코드에서는 `let`이나 `const`를 사용하세요.

## 데이터 타입

JavaScript의 주요 데이터 타입을 알아봅시다.

### 1. 문자열 (String)

텍스트 데이터입니다. 따옴표나 쌍따옴표로 감싸세요.

```javascript
let message = "Hello, JavaScript!";
let author = 'Velocibet';

console.log(message);  // "Hello, JavaScript!"
```

### 2. 숫자 (Number)

정수와 소수 모두 포함합니다.

```javascript
let count = 10;
let price = 19.99;
let temperature = -5;

console.log(count + 5);      // 15
console.log(price * 2);      // 39.98
```

### 3. 불값 (Boolean)

참(true) 또는 거짓(false) 중 하나입니다.

```javascript
let isActive = true;
let isEmpty = false;

if (isActive) {
  console.log("활성화되었습니다!");
}
```

### 4. null과 undefined

```javascript
let empty = null;          // 명시적으로 "값이 없음"
let notDefined;            // 선언만 함 (undefined)

console.log(empty);        // null
console.log(notDefined);   // undefined
```

## 변수명 규칙

변수명을 지을 때는 다음 규칙을 따라야 합니다.

```javascript
let userName = "Alice";      // 좋은 예 (camelCase)
let user_name = "Bob";       // 가능하지만 권장 안 함
let userName2 = "Charlie";   // 숫자 포함 가능 (맨 앞은 안 됨)

// let 2user = "David";       // 오류! 숫자로 시작 불가
// let user-name = "Eve";     // 오류! 하이픈 사용 불가
```

**가독성을 위해 camelCase를 사용하세요**: `userName`, `emailAddress`, `isLoggedIn`

## 실제 예제

간단한 계산 프로그램을 만들어봅시다.

```javascript
// 사람의 정보 저장
const name = "Velocibet";
const birthYear = 2000;

// 현재 연도
const currentYear = 2026;

// 나이 계산
let age = currentYear - birthYear;

console.log(name + "은 " + age + "살입니다.");
// 출력: Velocibet은 26살입니다.
```

또는 더 간편하게:

```javascript
const name = "Velocibet";
const age = 26;

console.log(`${name}은 ${age}살입니다.`);
// 출력: Velocibet은 26살입니다. (백틱 사용)
```

## 변수 사용 시 주의사항

- 변경되지 않을 값은 `const`로, 변경할 값은 `let`으로 선언하세요
- 변수명은 명확하게 작성하세요 (`x` 보다는 `userName`)
- 같은 이름의 변수를 여러 번 선언하지 마세요
