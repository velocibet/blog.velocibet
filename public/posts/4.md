## 링크 만들기 – a 태그

웹 페이지에서 다른 페이지로 이동하게 하거나 특정 URL로 연결할 때는 `<a>` 태그를 사용합니다.  
`a`는 **anchor(닻)**의 약자로, 웹에서 흔히 “링크”라고 부르는 요소입니다.

기본 구조는 이렇게 생겼습니다.

```html
<a href="https://example.com">예시 사이트로 이동</a>
```

- `href`: 이동할 URL을 지정합니다.
- 링크 텍스트를 클릭하면 브라우저가 href에 적힌 주소로 이동합니다.
- 필요하다면 `target="_blank"` 속성을 추가해 새 탭에서 열 수도 있습니다.

```html
<a href="https://example.com" target="_blank">새 탭에서 열기</a>
```

## 이미지 삽입 – img 태그

웹 페이지에 이미지를 보여주고 싶을 때는 `<img>` 태그를 사용합니다.
`img`는 **image(이미지)**의 약자입니다. 이 태그는 닫는 태그가 필요 없는 self-closing 태그예요.

기본 구조는 다음과 같습니다.

```html
<img src="image.png" alt="이미지 설명">
```

- `src`: 보여줄 이미지 파일의 경로를 지정합니다.
- `alt`: 이미지가 표시되지 않을 때 대신 보여줄 텍스트, 접근성에도 중요합니다.
- 필요하다면 `width`나 `height` 속성을 추가해 크기를 조절할 수 있습니다.

```html
<img src="image.png" alt="풍경 사진" width="300">
```

링크와 이미지는 웹 페이지에서 가장 자주 쓰이는 요소 중 하나입니다.
HTML을 배우면서 이 두 가지 태그의 구조와 속성을 잘 이해해두면, 페이지를 만들 때 콘텐츠를 자유롭게 연결하고 이미지를 표시할 수 있습니다.
