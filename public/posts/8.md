## CSS를 HTML에 연결하기

CSS를 사용하려면 먼저 HTML 파일과 CSS 파일을 연결해야 합니다.

HTML 파일의 `<head>` 태그 안에 다음과 같이 작성합니다:

```html
<head>
  <link rel="stylesheet" href="style.css">
</head>
```

- `rel="stylesheet"`: 이것이 스타일 시트라고 지정
- `href="style.css"`: 연결할 CSS 파일의 경로

## 자주 사용하는 CSS 속성들

### 색상 관련

```css
h1 {
  color: red;              /* 글자 색상 */
  background-color: yellow; /* 배경색 */
}
```

색상은 세 가지 방식으로 표현할 수 있습니다:
- 색상명: `red`, `blue`, `green`
- 16진수: `#FF0000`, `#0000FF`
- RGB: `rgb(255, 0, 0)`

### 텍스트 관련

```css
p {
  font-size: 18px;        /* 글자 크기 */
  font-family: Arial;     /* 글꼴 */
  font-weight: bold;      /* 글자 굵기 (normal, bold) */
  text-align: center;     /* 정렬 (left, center, right) */
  line-height: 1.5;       /* 줄 높이 */
}
```

### 여백 관련

```css
div {
  margin: 20px;           /* 바깥쪽 여백 */
  padding: 10px;          /* 안쪽 여백 */
  border: 1px solid black; /* 테두리 */
}
```

## 실제 예제

HTML:

```html
<h1>제목입니다</h1>
<p>이것은 문단입니다.</p>
```

CSS:

```css
h1 {
  color: blue;
  font-size: 32px;
  text-align: center;
  margin-bottom: 20px;
}

p {
  color: #333333;
  font-size: 16px;
  line-height: 1.6;
}
```

결과:
- 제목은 파란색, 32px 크기, 가운데 정렬됨
- 문단은 진회색, 16px 크기, 줄 간격이 넓음

## 카스케이딩(Cascading)이란?

CSS는 "Cascading" Style Sheets입니다. 이것은 여러 스타일이 적용될 때 **마지막에 작성된 스타일이 우선**된다는 뜻입니다.

```css
p {
  color: red;
}

p {
  color: blue;  /* 이것이 최종 적용됨 */
}
```

위의 경우 문단이 파란색으로 표시됩니다.

이제 기본적인 CSS 속성들을 알았으니, 다음 강좌에서는 **선택자(Selector)**에 대해 자세히 배워보겠습니다.
